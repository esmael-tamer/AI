# منصة AI - صفحة الويب الرئيسية

صفحة ويب احترافية وحديثة للمنصة الذكية مع تأثيرات بصرية متقدمة وتصميم responsive.

## 📋 المحتويات

### الملفات الرئيسية
- **landing.html** - الهيكل الأساسي والمحتوى
- **landing.css** - الأنماط والحركات المتقدمة (1900+ سطر)
- **landing.js** - التفاعلات والتأثيرات الديناميكية

## 🎯 الأقسام الرئيسية

### 1. شريط التنقل (Navigation Bar)
- ثابت في أعلى الصفحة
- تأثير backdrop blur
- روابط سلسة للأقسام المختلفة
- زر CTA أزرق مشع

### 2. القسم الرئيسي (Hero Section)
- عنوان كبير جذاب
- نص فرعي وصفي
- زرين للإجراء (CTA buttons)
- صورة معاينة بتأثير 3D
- خلفية ديناميكية بتدرجات وموجات
- إحصائيات المنصة

### 3. قسم المميزات (Features Section)
- 6 بطاقات مميزات
- أيقونات رموز تعبيرية
- تأثيرات hover متقدمة
- وصف مختصر لكل مميزة

### 4. قسم الأسعار (Pricing Section)
- 3 خطط تسعير
- خطة مميزة مع badge خاص
- قائمة المميزات لكل خطة
- أزرار CTA مختلفة

### 5. معرض الأعمال (Gallery Section)
- 6 مشاريع عينة
- صور بتدرجات ملونة
- تأثيرات hover وتكبير
- وصف لكل مشروع

### 6. قسم التواصل (Contact Section)
- معلومات الاتصال
- نموذج تواصل تفاعلي
- حقول إدخال مع validation
- رسالة نجاح عند الإرسال

### 7. التذييل (Footer)
- معلومات عن المنصة
- روابط سريعة
- روابط التواصل الاجتماعي
- حقوق الملكية

## 🚀 كيفية الاستخدام

### على Replit
1. أنشئ مشروع جديد "HTML, CSS, JS"
2. انسخ محتوى landing.html
3. انسخ محتوى landing.css
4. انسخ محتوى landing.js
5. اضغط Run

### على خادم ويب
1. انسخ الملفات الثلاثة
2. ارفعها على الخادم
3. افتح landing.html في المتصفح
